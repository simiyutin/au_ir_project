     <h1 id="ROS_Tutorials">ROS Tutorials</h1> 
     <span class="anchor" id="line-2"></span>
     <p class="line867"><strong>Non-Beginners</strong>: If you're already familiar enough with <tt class="backtick">ROS</tt> <a href="/fuerte">fuerte</a> or earlier versions and only want to explore the new build system introduced in <a href="/groovy">groovy</a> and used in <a href="/hydro">hydro</a> and later, called <a href="/catkin">catkin</a>, you can go through more in-depth <a href="/catkin/Tutorials">catkin tutorial here</a>. However, going over all basic <a href="/ROS/Tutorials#Beginner_Level">Beginner Level</a> tutorials is still recommended for all users to get exposed to new features. <span class="anchor" id="line-3"></span><span class="anchor" id="line-4"></span></p>
     <p class="line867"><strong>If you are new to Linux</strong>: You may find it helpful to first do a quick tutorial on common command line tools for linux. A good one is <a class="http" href="http://www.ee.surrey.ac.uk/Teaching/Unix/">here</a>. <span class="anchor" id="line-5"></span><span class="anchor" id="line-6"></span></p>

     <h2 id="Core_ROS_Tutorials">Core ROS Tutorials</h2> 
     <span class="anchor" id="line-9"></span>
     <p class="line867"> </p>
     <h3 id="Beginner_Level">Beginner Level</h3> 
     <span class="anchor" id="line-10"></span>
     <p class="line867"></p>
     <ol>
      <li><a href="/ROS/Tutorials/InstallingandConfiguringROSEnvironment">Installing and Configuring Your ROS Environment</a><p><span class="anchor" id="line-1-1"></span>This tutorial walks you through installing ROS and setting up the ROS environment on your computer. </p></li>
      <li><a href="/ROS/Tutorials/NavigatingTheFilesystem">Navigating the ROS Filesystem</a><p><span class="anchor" id="line-1-2"></span>This tutorial introduces ROS filesystem concepts, and covers using the roscd, rosls, and <a href="/rospack">rospack</a> commandline tools. </p></li>
      <li><a href="/ROS/Tutorials/CreatingPackage">Creating a ROS Package</a><p><span class="anchor" id="line-1-3"></span>This tutorial covers using <a href="/roscreate">roscreate-pkg</a> or <a href="/catkin">catkin</a> to create a new package, and <a href="/rospack">rospack</a> to list package dependencies. </p></li>
      <li><a href="/ROS/Tutorials/BuildingPackages">Building a ROS Package</a><p><span class="anchor" id="line-1-4"></span>This tutorial covers the toolchain to build a package. </p></li>
      <li><a href="/ROS/Tutorials/UnderstandingNodes">Understanding ROS Nodes</a><p><span class="anchor" id="line-1-5"></span>This tutorial introduces ROS graph concepts and discusses the use of <a href="/roscore">roscore</a>, <a href="/rosnode">rosnode</a>, and <a href="/rosrun">rosrun</a> commandline tools. </p></li>
      <li><a href="/ROS/Tutorials/UnderstandingTopics">Understanding ROS Topics</a><p><span class="anchor" id="line-1-6"></span>This tutorial introduces ROS topics as well as using the <a href="/rostopic">rostopic</a> and <a href="/rqt_plot">rqt_plot</a> commandline tools. </p></li>
      <li><a href="/ROS/Tutorials/UnderstandingServicesParams">Understanding ROS Services and Parameters</a><p><span class="anchor" id="line-1-7"></span>This tutorial introduces ROS services, and parameters as well as using the <a href="/rosservice">rosservice</a> and <a href="/rosparam">rosparam</a> commandline tools. </p></li>
      <li><a href="/ROS/Tutorials/UsingRqtconsoleRoslaunch">Using rqt_console and roslaunch</a><p><span class="anchor" id="line-1-8"></span>This tutorial introduces ROS using <a href="/rqt_console">rqt_console</a> and <a href="/rqt_logger_level">rqt_logger_level</a> for debugging and <a href="/roslaunch">roslaunch</a> for starting many nodes at once. If you use <tt class="backtick">ROS&nbsp;fuerte</tt> or ealier distros where <a href="/rqt">rqt</a> isn't fully available, please see this page with <a href="/ROS/Tutorials/UsingRxconsoleRoslaunch">this page</a> that uses old <tt class="backtick">rx</tt> based tools. </p></li>
      <li><a href="/ROS/Tutorials/UsingRosEd">Using rosed to edit files in ROS</a><p><span class="anchor" id="line-1-9"></span>This tutorial shows how to use <a href="/rosbash">rosed</a> to make editing easier. </p></li>
      <li><a href="/ROS/Tutorials/CreatingMsgAndSrv">Creating a ROS msg and srv</a><p><span class="anchor" id="line-1-10"></span>This tutorial covers how to create and build msg and srv files as well as the <a href="/rosmsg">rosmsg</a>, rossrv and roscp commandline tools. </p></li>
      <li><a href="/ROS/Tutorials/WritingPublisherSubscriber%28c%2B%2B%29">Writing a Simple Publisher and Subscriber (C++)</a><p><span class="anchor" id="line-1-11"></span>This tutorial covers how to write a publisher and subscriber node in C++. </p></li>
      <li><a href="/ROS/Tutorials/WritingPublisherSubscriber%28python%29">Writing a Simple Publisher and Subscriber (Python)</a><p><span class="anchor" id="line-1-12"></span>This tutorial covers how to write a publisher and subscriber node in python. </p></li>
      <li><a href="/ROS/Tutorials/ExaminingPublisherSubscriber">Examining the Simple Publisher and Subscriber</a><p><span class="anchor" id="line-1-13"></span>This tutorial examines running the simple publisher and subscriber. </p></li>
      <li><a href="/ROS/Tutorials/WritingServiceClient%28c%2B%2B%29">Writing a Simple Service and Client (C++)</a><p><span class="anchor" id="line-1-14"></span>This tutorial covers how to write a service and client node in C++. </p></li>
      <li><a href="/ROS/Tutorials/WritingServiceClient%28python%29">Writing a Simple Service and Client (Python)</a><p><span class="anchor" id="line-1-15"></span>This tutorial covers how to write a service and client node in python. </p></li>
      <li><a href="/ROS/Tutorials/ExaminingServiceClient">Examining the Simple Service and Client</a><p><span class="anchor" id="line-1-16"></span>This tutorial examines running the simple service and client. </p></li>
      <li><a href="/ROS/Tutorials/Recording%20and%20playing%20back%20data">Recording and playing back data</a><p><span class="anchor" id="line-1-17"></span>This tutorial will teach you how to record data from a running ROS system into a .bag file, and then to play back the data to produce similar behavior in a running system. </p></li>
      <li><a href="/ROS/Tutorials/Getting%20started%20with%20roswtf">Getting started with roswtf</a><p><span class="anchor" id="line-1-18"></span>Basic introduction to the <a href="/roswtf">roswtf</a> tool. </p></li>
      <li><a href="/ROS/Tutorials/NavigatingTheWiki">Navigating the ROS wiki</a><p><span class="anchor" id="line-1-19"></span>This tutorial discusses the layout of the ROS wiki (<a href="/Documentation">ros.org</a>) and talks about how to find what you want to know. </p></li>
      <li><a href="/ROS/Tutorials/WhereNext">Where Next?</a><p><span class="anchor" id="line-1-20"></span>This tutorial discusses options for getting to know more about using ROS on real or simulated robots. </p></li>
     </ol>

     <h3 id="Intermediate_Level">Intermediate Level</h3> 
     <span class="anchor" id="line-15"></span>
     <p class="line862">More client API tutorials can be found in the relevant package (<a href="/roscpp/Tutorials">roscpp</a>, <a href="/rospy/Tutorials">rospy</a>, <a href="/roslisp/Tutorials">roslisp</a>) <span class="anchor" id="line-16"></span><span class="anchor" id="line-17"></span></p>
     <p class="line867"></p>
     <ol>
      <li><a href="/ROS/Tutorials/Creating%20a%20Package%20by%20Hand">Creating a ROS package by hand.</a><p><span class="anchor" id="line-1-21"></span>This tutorial explains how to manually create a ROS package. </p></li>
      <li><a href="/ROS/Tutorials/rosdep">Managing System dependencies</a><p><span class="anchor" id="line-1-22"></span>This explains how to use <a href="/rosdep">rosdep</a> to install system dependencies. </p></li>
      <li><a href="/ROS/Tutorials/Roslaunch%20tips%20for%20larger%20projects">Roslaunch tips for large projects</a><p><span class="anchor" id="line-1-23"></span>This tutorial describes some tips for writing roslaunch files for large projects. The focus is on how to structure launch files so they may be reused as much as possible in different situations. We'll use the 2dnav_pr2 package as a case study. </p></li>
      <li><a href="/ROS/Tutorials/MultipleMachines">Running ROS across multiple machines</a><p><span class="anchor" id="line-1-24"></span>This tutorial explains how to start a ROS system using two machines. It explains the use of <tt class="backtick">ROS_MASTER_URI</tt> to configure multiple machines to use a single master. </p></li>
      <li><a href="/ROS/Tutorials/DefiningCustomMessages">Defining Custom Messages</a><p><span class="anchor" id="line-1-25"></span>This tutorial will show you how to define your own custom message data types using the ROS <a href="/ROS/Message_Description_Language">Message Description Language</a>. </p></li>
      <li><a href="/ROS/Tutorials/Using%20a%20C%2B%2B%20class%20in%20Python">Using a C++ class in Python</a><p><span class="anchor" id="line-1-26"></span>This tutorial illustrates a way to use a C++ class with ROS messages in Python. </p></li>
      <li><a href="/ROS/Tutorials/Packaging%20your%20ROS%20project%20as%20a%20snap">Packaging your ROS project as a snap</a><p><span class="anchor" id="line-1-27"></span>This tutorial covers how to package and deploy your ROS project as a snap. </p></li>
      <li><a href="/WritingTutorials">How to Write a Tutorial</a><p><span class="anchor" id="line-1-28"></span>This tutorial covers useful template and macros for writing tutorials, along with example tutorials that are available for guidance on <a href="/Documentation">ros.org</a> </p></li>
     </ol>
